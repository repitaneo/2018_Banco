# 2018_Banco
Proyecto que recrea cuentas en un banco
Permite:
* Ingresos
* Reintegros
* Transferencias

Además cuenta con 
* setters
* getters
* toString

## Requiere
* Tener instalado Java y Eclipse

## ¿Qué descargas? 
* Un proyecto de Eclipse que podrás importar a tu workspacer desde File > Import > General > Import Files intro workspace
